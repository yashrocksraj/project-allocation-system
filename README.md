# Project Allocation System (Flask Backend)

This is a backend REST API built using **Flask** and **SQLAlchemy** that manages:

- Users
- Projects
- Assignment of users to projects
- Prevention of duplicate project assignments

The project demonstrates backend fundamentals such as:
- REST APIs
- Database modeling
- CRUD operations
- Error handling
- Git & GitHub usage

---

## Tech Stack

- Python 3
- Flask
- Flask-SQLAlchemy
- SQLite
- Git & GitHub

---

## Features

- Create users
- Create projects
- Assign users to projects
- Prevent duplicate user–project assignments
- Update user email
- Fetch users and projects
